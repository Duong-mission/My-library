Lazy Loading trong React: Tối Ưu Tải Trang

Tải các phần ứng dụng khi cần thiết với `React.lazy` và `Suspense`.

Lazy Loading (Code Splitting)

Tải động các Module

  • Kỹ thuật chia nhỏ code base thành các gói (bundle) nhỏ hơn.
  • Chỉ tải những phần của ứng dụng khi chúng thực sự cần thiết.
  • Giảm kích thước bundle JavaScript ban đầu.

Triển khai với `React.lazy` & `Suspense`

Chia nhỏ Code Base Hiệu Quả

  • `React.lazy()`: Hàm cho phép render một component được import động.
  • `Suspense`: Component bao bọc `React.lazy` để hiển thị UI dự phòng (fallback) trong khi component đang tải.
  • Cú pháp: `<Suspense fallback={<div>Loading...</div>}><LazyComponent /></Suspense>`

Lợi Ích Chính

Cải thiện Trải nghiệm người dùng

  • **Giảm thời gian tải trang ban đầu:** Người dùng thấy nội dung nhanh hơn.
  • **Tối ưu tài nguyên:** Chỉ tải code cần thiết, tiết kiệm băng thông.
  • **Cải thiện hiệu suất:** Giảm thời gian parsing và execution JavaScript.
  • Hỗ trợ tốt cho các ứng dụng lớn và phức tạp.
