React Context API: Quản lý State Hiệu quả

Giải pháp tích hợp cho việc chia sẻ dữ liệu xuyên suốt các component.

Context API là gì?

Giải pháp quản lý State tích hợp

  • Là cơ chế chính thức của React để truyền dữ liệu "xuống" cây component mà không cần truyền props thủ công qua mọi cấp.
  • Giải quyết vấn đề "prop drilling" (truyền props qua nhiều cấp trung gian).
  • Được thiết kế để chia sẻ dữ liệu "global" cho một cây component.

Khi nào sử dụng Context API?

Các trường hợp lý tưởng

  • Dữ liệu Global: Thông tin cần truy cập bởi nhiều component ở các cấp khác nhau.
  • Chủ đề (Theme): Quản lý chế độ sáng/tối hoặc bảng màu ứng dụng.
  • Xác thực người dùng (User Authentication): Trạng thái đăng nhập, thông tin người dùng.
  • Ngôn ngữ (Localization): Quản lý ngôn ngữ hiển thị của ứng dụng.

Cách tạo Context

Provider & Consumer (hoặc useContext)

  • 1. Tạo Context: const MyContext = React.createContext(defaultValue);
  • 2. Cung cấp giá trị (Provider): <MyContext.Provider value={...}>
  • 3. Sử dụng giá trị (Consumer):
    • Class Components: <MyContext.Consumer>{value => ...}</MyContext.Consumer>
    • Function Components (recommended): const value = useContext(MyContext);
