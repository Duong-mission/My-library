Styled Components: CSS-in-JS cho React

Tích hợp CSS mạnh mẽ trực tiếp vào các component React của bạn.

Styled Components là gì?

Viết CSS hiện đại trong JavaScript.

  • Thư viện CSS-in-JS phổ biến.
  • Cho phép bạn viết CSS trực tiếp trong các file JavaScript/TypeScript của React.
  • Tạo ra các component React với style riêng biệt, đóng gói.

Lợi ích chính

Tăng cường tính linh hoạt và khả năng bảo trì.

  • Dynamic Styling: Thay đổi style dựa trên props hoặc state của component.
  • Scoped CSS: Tránh xung đột style giữa các component nhờ tên class tự động.
  • Dễ bảo trì: Style liên kết trực tiếp với logic component, dễ dàng quản lý.
  • Tối ưu hóa: Chỉ tải CSS cần thiết cho các component được render.

Cách tạo và sử dụng

Đơn giản hóa quá trình styling component.

  • Bước 1: Import styled: import styled from 'styled-components';
  • Bước 2: Định nghĩa component: const Button = styled.button
  • Bước 3: Viết CSS: `background: #3F51B5; color: white; padding: 10px;`
  • Bước 4: Sử dụng: <Button>Click Me</Button> (như một component React bình thường).
