Functional Components & React Hooks

Nâng cấp phát triển UI với Functional Components và sức mạnh của Hooks.

Functional Components

Hướng tiếp cận mới trong React

  • Phương pháp được khuyến nghị từ React v16.8 (2018).
  • Được khai báo dưới dạng hàm JavaScript nhận `props` và trả về JSX.
  • Đơn giản, dễ đọc và dễ kiểm thử hơn Class Components.

Định nghĩa React Hooks

Mở rộng khả năng của Functional Components

  • Các hàm đặc biệt cho phép "hook vào" các tính năng của React.
  • Sử dụng state và lifecycle trong Functional Components mà không cần class.
  • Ví dụ: `useState` (quản lý state), `useEffect` (xử lý side effects).

Ưu điểm vượt trội của Hooks

So với Class Components truyền thống

  • **Loại bỏ Class:** Không cần `this` và các vấn đề ràng buộc.
  • **Tái sử dụng logic:** Dễ dàng chia sẻ logic stateful giữa các components.
  • **Code sạch hơn:** Giảm độ phức tạp, tăng tính dễ đọc.
  • **Tối ưu hiệu suất:** Tránh các vấn đề của Class Components.
