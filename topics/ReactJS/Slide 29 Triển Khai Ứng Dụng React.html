Triển Khai Ứng Dụng React

Đưa ứng dụng web của bạn ra môi trường thực tế

1. Chuẩn Bị Ứng Dụng (Build)

Quy trình đóng gói cho sản phẩm

  • Chạy lệnh `npm run build` hoặc `yarn build`.
  • Tạo phiên bản mã nguồn đã được tối ưu hóa.
  • Kết quả là thư mục `build` (hoặc `dist`).
  • Chứa HTML, CSS, JavaScript đã được nén và tối ưu.

2. Nền Tảng Triển Khai Phổ Biến

Các dịch vụ Hosting hàng đầu

  • Vercel & Netlify: Triển khai cực nhanh, tích hợp CI/CD, CDN.
  • Firebase Hosting: Của Google, tích hợp sâu các dịch vụ backend.
  • GitHub Pages: Miễn phí, lý tưởng cho dự án cá nhân, mã nguồn mở.
  • AWS S3/CloudFront: Linh hoạt, mạnh mẽ, cho quy mô lớn và tùy biến.

3. Cấu Hình Triển Khai Cơ Bản

Những cài đặt cần thiết

  • Thư mục gốc: Chọn `build` (hoặc `dist`) làm thư mục gốc.
  • SPA Fallback: Cấu hình rewrite mọi request về `index.html`.
  • Biến môi trường: Thiết lập các biến `REACT_APP_...` trên nền tảng.
  • Tên miền tùy chỉnh: Kết nối tên miền riêng của ứng dụng.
