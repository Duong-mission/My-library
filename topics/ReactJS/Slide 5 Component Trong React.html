React Components: Nền tảng xây dựng UI

Hiểu về các khối xây dựng cơ bản của ứng dụng React

Component là gì?

Khối xây dựng cơ bản của UI

  • Là các khối UI độc lập, có thể tái sử dụng.
  • Kết hợp các component để tạo nên giao diện người dùng phức tạp.
  • Mỗi component quản lý trạng thái (state) và logic riêng của nó.
  • Giúp code dễ quản lý, bảo trì và phát triển.

Phân loại Component

Class Components và Functional Components

  • Class Components:

    • Sử dụng cú pháp ES6 class.
    • Quản lý state và lifecycle bằng các phương thức (methods).

  • Functional Components:

    • Là các hàm JavaScript thuần túy.
    • Sử dụng React Hooks (useState, useEffect,...) từ v16.8+.
    • Được khuyến nghị sử dụng cho các dự án mới.

Cấu trúc Component

Ví dụ Functional Component đơn giản

  • Định nghĩa:

    function Welcome(props) {
      return <h1>Hello, {props.name}!</h1>;
    }

  • Sử dụng JSX:

    <Welcome name="World" />

  • Đầu ra:

    Hello, World!
