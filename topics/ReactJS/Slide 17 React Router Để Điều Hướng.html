Quản lý Điều hướng trong React: Giới thiệu React Router

Khám phá thư viện chuẩn để xây dựng Single-Page Applications (SPA) với React.

React Router: Nền Tảng Điều Hướng

Giải pháp routing chuẩn cho React SPA.

  • React không có tính năng routing tích hợp sẵn.
  • React Router là thư viện bên thứ ba phổ biến và được khuyến nghị.
  • Giúp định nghĩa routes, quản lý điều hướng và xử lý thay đổi URL.
  • Thiết yếu cho việc xây dựng các Single-Page Applications (SPA) hiệu quả.

Các Khái Niệm Cơ Bản

Hiểu các building blocks của React Router.

  • Router: Thành phần bao bọc toàn bộ ứng dụng để kích hoạt cơ chế routing.
    • BrowserRouter: Phổ biến nhất cho ứng dụng web.
  • Route: Định nghĩa đường dẫn URL và component sẽ hiển thị khi đường dẫn đó khớp.
    • Sử dụng thuộc tính path và element.
  • Link: Tạo liên kết điều hướng giữa các route mà không tải lại toàn bộ trang.
    • Tương tự thẻ `` nhưng được tối ưu cho SPA.

Thiết Lập Điều Hướng Cơ Bản

Các bước khởi tạo routing trong ứng dụng React.

  • Cài đặt thư viện: `npm install react-router-dom`.
  • Bọc App Component của bạn bằng BrowserRouter.
  • Sử dụng thành phần Routes để nhóm tất cả các Route.
  • Trong Routes, định nghĩa từng Route với path (đường dẫn) và element (component).
  • Dùng thành phần Link để tạo các liên kết điều hướng nội bộ đến các path đã định nghĩa.
