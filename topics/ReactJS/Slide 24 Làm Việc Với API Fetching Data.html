Gọi API trong ReactJS

Tích hợp dữ liệu từ bên ngoài vào ứng dụng của bạn

Phương thức gọi API

Sử dụng `fetch` hoặc `axios`

  • `fetch` API: Được tích hợp sẵn trong trình duyệt, đơn giản để sử dụng mà không cần cài đặt thêm.
  • `axios`: Thư viện bên thứ ba phổ biến, cung cấp nhiều tính năng mạnh mẽ (interceptors, tự động chuyển đổi JSON).
  • Thực hiện cuộc gọi trong `useEffect` (khi component mount) hoặc trong các hàm xử lý sự kiện.

Xử lý dữ liệu & Cập nhật State

Quản lý trạng thái với Hooks

  • `useState`: Dùng để lưu trữ dữ liệu trả về từ API, trạng thái loading và lỗi.
  • `useEffect`: Thực hiện cuộc gọi API khi component được render lần đầu. Đảm bảo clean-up nếu cần.
  • Xử lý phản hồi (response) từ API (chuyển đổi JSON, kiểm tra lỗi) và cập nhật state tương ứng.
  • Quản lý các trạng thái: `isLoading`, `isError`, `data`.

Hiển thị dữ liệu từ API

Trình bày thông tin động

  • Sử dụng các câu điều kiện để hiển thị trạng thái `Loading...` khi đang fetch dữ liệu.
  • Hiển thị thông báo `Error!` nếu có lỗi xảy ra trong quá trình gọi API.
  • Khi dữ liệu có sẵn, sử dụng phương thức `map()` để render danh sách các item một cách động.
  • Đảm bảo giao diện người dùng (UI) phản hồi linh hoạt với các thay đổi của dữ liệu.
