Styling trong React: Khám Phá CSS Modules

Nâng cao hiệu quả quản lý CSS với các phương pháp hiện đại

Các Phương Pháp Styling

Đa dạng lựa chọn cho giao diện

  • CSS truyền thống (.css)
  • Inline CSS (style attribute)
  • CSS-in-JS (Styled Components, Emotion)
  • CSS Preprocessors (Sass, Less)
  • CSS Modules (trọng tâm chính)

CSS Modules: CSS Cục Bộ

Giải pháp cho xung đột tên class

  • Tự động tạo tên class độc nhất (ví dụ: `App_title__abc12`)
  • Ngăn chặn hoàn toàn xung đột CSS toàn cục
  • Đảm bảo style chỉ áp dụng cho component đó
  • Giúp dễ dàng bảo trì và tái sử dụng component
  • Mỗi file `.module.css` là một module riêng biệt

Cách Sử Dụng CSS Modules

Import và áp dụng trong Component

  • Đặt tên file: ComponentName.module.css
  • Import: import styles from './MyComponent.module.css';
  • Áp dụng: <div className={styles.myClass}>
  • Cấu hình build (Webpack, Vite) hỗ trợ sẵn
  • Tích hợp mượt mà với Create React App, Next.js
