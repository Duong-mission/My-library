JSX: Sức Mạnh Của JavaScript XML

Hiểu về cú pháp nền tảng cho React UI

JSX là gì?

JavaScript XML

  • Là phần mở rộng cú pháp cho JavaScript.
  • Cho phép viết mã giống HTML trong các tệp JavaScript.
  • Không phải HTML thật, được chuyển đổi bởi transpiler (ví dụ: Vite) thành JavaScript thông thường.

Tại sao React sử dụng JSX?

Trực quan & Hiệu quả

  • Giúp viết giao diện người dùng (UI) trực quan, dễ đọc như HTML.
  • Cho phép kết hợp logic JavaScript và markup HTML một cách liền mạch.
  • Làm cho code component dễ hiểu, dễ bảo trì và có thể mở rộng tốt hơn.

Cú pháp cơ bản

Thẻ, thuộc tính & Biểu thức

  • Thẻ đóng: Mọi thẻ phải được đóng (ví dụ: `<img />` hoặc `<div></div>`).
  • Thuộc tính `className`: Thay thế cho thuộc tính `class` trong HTML.
  • Nhúng biểu thức JavaScript: Sử dụng dấu ngoặc nhọn `{}` để chèn biến, hàm, v.v.
