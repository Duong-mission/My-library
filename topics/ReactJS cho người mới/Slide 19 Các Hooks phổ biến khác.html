Các Hooks Nâng Cao Trong React

Mở rộng khả năng quản lý trạng thái và tương tác với DOM

useContext

Chia sẻ trạng thái dễ dàng

  • Chia sẻ state qua cây component mà không cần truyền props thủ công.

useRef

Truy cập DOM và giá trị không thay đổi

  • Truy cập trực tiếp các phần tử DOM hoặc giữ giá trị thay đổi không gây re-render.

useReducer

Quản lý trạng thái phức tạp

  • Quản lý state phức tạp hơn `useState`.
