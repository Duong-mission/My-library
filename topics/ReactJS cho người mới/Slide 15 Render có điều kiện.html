Conditional Rendering trong ReactJS

Các phương pháp hiển thị nội dung dựa trên điều kiện

1. `if/else` Statement

Điều kiện cơ bản

  • Phương pháp truyền thống trong JavaScript.
  • Thường sử dụng bên ngoài JSX.
  • Ứng dụng: Quyết định toàn bộ nội dung render.
  • Ví dụ: 
    if (isLoggedIn) {
  return <UserPanel />;
} else {
  return <LoginScreen />;
}

2. Toán tử ba ngôi `? :`

Render luân phiên

  • Cú pháp ngắn gọn, dùng trực tiếp trong JSX.
  • Cho phép render một trong hai phần tử.
  • Ứng dụng: Hiển thị hai trạng thái khác nhau.
  • Ví dụ: 
    {isLoggedIn ? (
  <WelcomeMessage />
) : (
  <GuestMessage />
)}

3. Toán tử logic `&&`

Render có điều kiện rút gọn

  • Dùng khi chỉ cần render nếu điều kiện đúng.
  • Nếu điều kiện là `false`, React sẽ bỏ qua.
  • Ứng dụng: Hiển thị thông báo, icon khi có dữ liệu.
  • Ví dụ: 
    {itemCount > 0 && (
  <p>Bạn có {itemCount} mục.</p>
)}
