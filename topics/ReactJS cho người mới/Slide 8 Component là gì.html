Component trong ReactJS: Khối Xây Dựng Cốt Lõi

Hiểu về cấu trúc, nguyên lý và các loại Component cơ bản.

Component là gì?

Khối xây dựng cơ bản

  • Mỗi ứng dụng React là tập hợp các phần giao diện (UI) độc lập.
  • Đại diện cho những phần tử UI có thể tái sử dụng.
  • Giúp code dễ đọc, dễ bảo trì và mở rộng.

Nguyên lý Modular Design

Độc lập & Tái sử dụng

  • Chia nhỏ UI phức tạp thành các phần nhỏ hơn, dễ quản lý.
  • Mỗi component có chức năng riêng, không ảnh hưởng các phần khác.
  • Tối ưu hóa hiệu suất thông qua cơ chế Virtual DOM.

Các loại Component

Tập trung vào Functional Component

  • **Functional Component:** Hàm JavaScript nhận `props`, trả về JSX (phương pháp hiện đại, khuyến nghị).
  • **Class Component:** Lớp JavaScript kế thừa `React.Component` (phương pháp cũ hơn).
  • Người mới nên ưu tiên học Functional Component và React Hooks.
