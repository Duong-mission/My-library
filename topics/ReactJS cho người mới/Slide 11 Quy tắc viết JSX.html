Các Quy Ước Code Cơ Bản trong ReactJS

Tối ưu hóa khả năng đọc và bảo trì ứng dụng của bạn

Phần tử Gốc Duy Nhất

Đảm bảo cấu trúc Component rõ ràng

  • Mỗi component phải trả về một phần tử gốc duy nhất (single root element).
  • Điều này giúp React quản lý và cập nhật DOM ảo một cách hiệu quả.
  • Có thể sử dụng `` hoặc `<>` để nhóm các phần tử mà không thêm node DOM thừa.

Sử dụng className

Tránh xung đột từ khóa JavaScript

  • Dùng thuộc tính `className` thay vì `class` khi định nghĩa các lớp CSS trong JSX.
  • `class` là một từ khóa dành riêng trong JavaScript, việc sử dụng nó có thể gây lỗi cú pháp.
  • `className` hoạt động tương tự như `class` trong HTML sau khi được biên dịch.

Quy Tắc JSX

Tuân thủ cú pháp JavaScript và HTML

  • Sử dụng `camelCase` cho các thuộc tính HTML/DOM trong JSX, ví dụ: `onClick`, `htmlFor`, `tabIndex`.
  • Nhúng biểu thức JavaScript (biến, hàm, đối tượng) trực tiếp vào trong JSX bằng cách sử dụng cặp dấu ngoặc nhọn `{}`.
  • Điều này cho phép kết hợp logic JavaScript một cách liền mạch với cấu trúc giao diện.
