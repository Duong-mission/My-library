React Router: Xây dựng ứng dụng SPA đa trang

Điều hướng mượt mà và hiệu quả trong các ứng dụng React.

Tại sao cần React Router?

Xây dựng ứng dụng đơn trang (SPA)

  • Ứng dụng một trang duy nhất nhưng có nhiều "view".
  • Chuyển đổi nội dung mà không tải lại toàn bộ trang.
  • Cung cấp trải nghiệm người dùng liền mạch và nhanh chóng.

Cài đặt & Component chính

Bắt đầu với React Router DOM

  • Cài đặt: npm install react-router-dom.
  • BrowserRouter: Bao bọc toàn bộ ứng dụng để kích hoạt định tuyến.
  • Routes: Chứa các định nghĩa Route.
  • Route: Ánh xạ đường dẫn URL tới component.
  • Link: Tạo liên kết điều hướng (thay thế thẻ ``).

Ví dụ: Trang Home & About

Thiết lập các trang cơ bản

  • Tạo component HomePage.jsx.
  • Tạo component AboutPage.jsx.
  • Sử dụng <Link to="/about"> để điều hướng.
  • Định nghĩa <Route path="/about" element={<AboutPage />} />.
