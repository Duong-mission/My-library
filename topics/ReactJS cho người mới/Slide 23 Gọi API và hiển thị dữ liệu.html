Tương Tác API trong ReactJS

Tích hợp `fetch`/Axios, `useEffect`, `useState` để xử lý API hiệu quả.

1. Gọi API

Sử dụng `fetch` hoặc Axios

  • `fetch` API: Tích hợp sẵn trong trình duyệt, đơn giản.
  • Axios: Thư viện bên thứ ba, nhiều tính năng hơn (interceptors, timeout).
  • Thực hiện các yêu cầu HTTP (GET, POST, PUT, DELETE).
  • Ví dụ: `fetch('url')` hoặc `axios.get('url')`.

2. Quản Lý Dữ Liệu

Kết hợp `useState` và `useEffect`

  • `useState`: Lưu trữ dữ liệu API, trạng thái loading/error.
  • `useEffect`: Thực hiện gọi API sau khi component render.
  • Sử dụng dependency array `[]` để chỉ gọi API một lần duy nhất.
  • Cập nhật state với dữ liệu nhận được từ API.

3. Hiển Thị & Xử Lý

Giao diện và Trạng thái

  • Hiển thị dữ liệu từ state lên giao diện người dùng.
  • Xử lý trạng thái `loading` (ví dụ: hiển thị spinner).
  • Xử lý trạng thái `error` (ví dụ: hiển thị thông báo lỗi).
  • Sử dụng Conditional Rendering để kiểm soát hiển thị.
