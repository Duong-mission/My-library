Quản Lý State Trong ReactJS

Hiểu về State và Hook useState

State Là Gì?

Dữ liệu nội bộ của Component

  • Dữ liệu riêng của một component.
  • Có thể thay đổi theo thời gian.
  • Khi State thay đổi, component sẽ tự động cập nhật lại giao diện (re-render).
  • Khác với Props (dữ liệu truyền từ cha xuống con, không thay đổi nội bộ).

`useState` Hook

Quản lý State trong Functional Component

  • Là một hàm đặc biệt từ React để quản lý state.
  • Cung cấp cách thức để component "ghi nhớ" dữ liệu.
  • Chỉ sử dụng trong Functional Component.
  • Cần import: `import { useState } from 'react';`

Cú Pháp & Ví Dụ Đếm

Khai báo và Cập nhật State

  • **Khai báo:** `const [value, setValue] = useState(initialValue);`
    • `value`: Giá trị state hiện tại.
    • `setValue`: Hàm để cập nhật state.
  • **Cập nhật:** Gọi `setValue(newValue)`.
  • **Ví dụ:** Bộ đếm đơn giản với `count` và `setCount`.
