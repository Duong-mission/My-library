Các Phương Pháp Styling trong ReactJS

Đa dạng lựa chọn để tạo giao diện đẹp và hiệu quả

Phương Pháp Styling Phổ Biến

Lựa chọn đa dạng cho UI

  • CSS Thông Thường: Sử dụng các tệp `.css` truyền thống. Đơn giản, dễ học và quen thuộc.
  • CSS Modules: Tự động phạm vi hóa (scope) CSS, giúp tránh xung đột tên lớp. Tên tệp thường là `.module.css`.
  • Styled Components: Viết CSS trực tiếp trong JavaScript bằng template literals (CSS-in-JS). Tăng tính linh hoạt và động.

Áp Dụng Inline Styles

Styling trực tiếp trong JSX

  • Cú pháp: Sử dụng đối tượng JavaScript `{ key: 'value' }` trong thuộc tính `style`.
  • Ví dụ: `
    ...
    `.
  • Ưu điểm: Nhanh chóng cho các style động, dễ dàng áp dụng cho từng phần tử cụ thể.
  • Nhược điểm: Khó bảo trì với nhiều style, không tận dụng được CSS cascade, không hỗ trợ pseudo-classes.

Import và Sử Dụng Tệp CSS

Quản lý style hiệu quả với CSS bên ngoài

  • Cách Import:
    • `import './App.css';` (CSS thông thường)
    • `import styles from './MyComponent.module.css';` (CSS Modules)
  • Áp dụng class: Sử dụng thuộc tính `className` thay vì `class` trong JSX.
  • Ví dụ: `
    ...
    ` hoặc `
    ...
    `.
  • Lưu ý: Đảm bảo tệp CSS được import vào component gốc hoặc file chính để các style được tải.
