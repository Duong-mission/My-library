Thiết Lập Môi Trường Phát Triển ReactJS

Các công cụ và bước cần thiết để bắt đầu dự án React đầu tiên của bạn.

Node.js và Quản Lý Gói

Nền tảng cho phát triển Front-end hiện đại

  • Node.js: Môi trường chạy JavaScript ngoài trình duyệt, cần thiết cho các công cụ xây dựng React.
  • npm (Node Package Manager): Trình quản lý gói mặc định của Node.js, dùng để cài đặt và quản lý các thư viện.
  • Yarn: Một lựa chọn thay thế npm, thường được biết đến với tốc độ và độ tin cậy.

Cài Đặt Node.js

Các bước để thiết lập hệ thống

  • Tải xuống: Truy cập trang web chính thức của Node.js và tải phiên bản LTS (Long Term Support) phù hợp với hệ điều hành của bạn.
  • Thực hiện cài đặt: Chạy file cài đặt và làm theo hướng dẫn trên màn hình. npm sẽ được cài đặt kèm theo.
  • Kiểm tra cài đặt: Mở Terminal/CMD và gõ node -v và npm -v để xác minh phiên bản.

Khởi Tạo & Chạy Ứng Dụng React

Bắt đầu dự án React đầu tiên của bạn

  • Create React App: Công cụ chính thức giúp khởi tạo dự án React mà không cần cấu hình phức tạp.
  • Tạo dự án mới: Trong Terminal/CMD, chạy lệnh npx create-react-app ten-du-an-cua-ban.
  • Vào thư mục dự án: Sau khi tạo xong, điều hướng vào thư mục bằng cd ten-du-an-cua-ban.
  • Chạy ứng dụng: Gõ npm start (hoặc yarn start) để khởi động máy chủ phát triển và xem ứng dụng trên trình duyệt.
